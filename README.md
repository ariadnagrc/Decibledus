# DECIBLEDUS
Programa diseñado para medir en tiempo real el nivel de voz del usuario, representándolo en una escala de 0 a 100 para facilitar su interpretación. Es especialmente útil en situaciones donde el uso de auriculares durante llamadas y la reproducción de sonido desde el PC, como al jugar, dificultan percibir el volumen real de la voz. Este programa ayuda a mantener un tono adecuado sin excederse, siendo también ideal para presentaciones y grabaciones.

### MANUAL

## CARACTERÍSTICAS PRINCIPALES
- **Medición en tiempo real**.
- **Gráfica visual superpuesta** en el modo minimizado para optimizar el espacio en pantalla.
- **Sensibilidad ajustable** manualmente en función de tus necesidades.
- **Tamaño configurable** de la gráfica del modo minimizado.

## CÓMO USAR DECIBLEDUS
1. Clona el repositorio o descarga y ejecuta el JAR.
2. Ajusta la sensibilidad del micrófono si es necesario.
3. Inicia el programa para activar la gráfica.
4. Habla y monitorea.
